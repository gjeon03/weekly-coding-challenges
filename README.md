# weekly-coding-challenges

- 문제는 각 주 폴더의 README.md에 작성
- 문제 작성후 updateReadme.js 파일을 실행해 root/README.md 를 업데이트 (중복되는 문제 확인 및 전체 문제를 보기 위해)
  ```
  node updateReadme.js
  ```
- 풀이는 각 주 폴더의 개인 폴더를 만들어 실행파일을 놓기
